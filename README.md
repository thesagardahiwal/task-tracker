# MERN Task Tracker - Production Ready

## Project Overview
A modern Task Tracker web application using the MERN stack.

## Tech Stack
### Frontend
- React 19, Vite, TypeScript, React Router v7
- Tailwind CSS v4, clsx, tailwind-merge
- TanStack Query, React Hook Form, Zod
- Axios, Sonner, Lucide React, Framer Motion

### Backend
- Node.js, Express.js, TypeScript
- MongoDB Atlas, Mongoose
- Zod, Helmet, Morgan, Compression, CORS, dotenv

## Architecture
- **Layered Backend Architecture:** Routes -> Controllers -> Services -> Data Access
- **Feature-Based Frontend Structure:** Grouping by domains (e.g., tasks) rather than types.
- **Strict Validation:** Using Zod on both ends.

## Git Workflow
We use Conventional Commits and a strict phase-by-phase development approach.
